# Base44

> Create custom eToro apps using Base44's no-code platform

Base44 is a powerful no-code development platform that partners with eToro to allow you to build custom trading applications and tools without writing extensive code.

<Note>
  We are currently rolling out Base44 integration to our users in stages. If you don't see this feature yet, stay tuned as we expand availability!
</Note>

## Getting Started

If you have access to Base44, please contact your **Account Manager** to help you create your first application.

<Steps>
  <Step title="Access eToro App Store">
    Navigate to the [eToro App Store](https://www.etoro.com/marketplace) to begin.
  </Step>
</Steps>

## Features

* **Visual Builder**: Design your app's interface and logic using Base44's drag-and-drop tools.
* **Native Integration**: Access eToro's real-time market data and portfolio features directly within Base44 components.
* **Vibe Coding**: Use natural language prompts to generate complex logic and workflows.
